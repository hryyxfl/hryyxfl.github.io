Centos7部署DHCP服务
一.安装
1.yum -y install DHCP;
二.配置文件
1./etc/dhcp/dhcpd.conf
2.配置文件为空，可以在其他文件（/usr/share/doc/dhcp*/dhcpd.conf.example例子),可以从其他文件中复制的到配置文件中。
3.配置内容
网段，地址范围，域名，网关，广播地址，默认租约时间，最大租约时间
4.增加一张网卡（仅主机），用于分配地址的，IP地址和网关地址可以一样
5.在开一台虚拟机，网卡仅主机，dhcp获取，(注意IPV4_FAILURE_FATAL【如果ipv4配置失败禁用设备】要开启）
6.重启dhcp,另一台虚拟机就能获取IP地址了。
